Wizualizacja Stylizacji List CSS

1. Różne Typy

  • Punktor: DISK
  • Element z kolorem tekstu.
  • Punktor: CIRCLE
  • Element z kolorem tekstu.
  • Punktor: SQUARE
  • Element z kolorem tekstu.

2. Punktor jako ZDJĘCIE

  • Punktor zastąpiony ikoną.
  • Drugi element.

3. Pozycja

  • Pozycja: INSIDE. Długi tekst zawija się POD punktor.
  • Kolejny element.
  • Pozycja: OUTSIDE. Długi tekst zawija się na wysokości tekstu.
  • Kolejny element.

4. Lista BEZ Punktora

  • Symbol usunięty.
  • Element listy w kolorze.

5. Skrócony Zapis (list-style)

  • Użyto skrótu.
  • Łączy w jednej linii typ, pozycję i obraz.
